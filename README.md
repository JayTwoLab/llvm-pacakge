# llvm-pacakge
- Pre-built LLVM Package
- Repository created for personal purposes.

## Environment variables
- Windows 11
   - `LLVM_HOME` : `C:\llvm-package`
   - `PATH` : `C:\llvm-package\bin`
   - `LLVM_DIR` : `C:\llvm-package\lib\cmake\llvm`
   - `Clang_DIR` : `C:\llvm-package\lib\cmake\clang`
- Linux
   - `LLVM_HOME` : `/home/jaytwo/workspace/llvm-package`
   - `PATH` : `"$LLVM_HOME/bin:$PATH"`
   - `LD_LIBRARY_PATH` : `"$LLVM_HOME/lib:$LD_LIBRARY_PATH"`
   - `CMAKE_PREFIX_PATH` : `"$LLVM_HOME:$CMAKE_PREFIX_PATH"`
   - `LLVM_DIR` : `"$LLVM_HOME/lib/cmake/llvm"`
   - `Clang_DIR` : `"$LLVM_HOME/lib/cmake/clang"`
